# Background-Changer-
A background changing website using html css and js 

# Live at :


## Features

* Change the background color with a button click
* Easy-to-use interface
* Beginner-friendly code
* Built with basic web technologies

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone this repository.
2. Open `index.html` in your web browser.
3. Click the buttons to change the background color.
